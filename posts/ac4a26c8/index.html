网络

光猫

  1. 桥接模式（Bridge）——光猫只负责光电转换，需要下接路由器

  2. 路由模式（Router）—— 光猫自己PPPoE 拨号，后面接的路由器，一般变成二级路由

路由器

拨号，桥接，路由

路由器最常用的核心模式有路由（Router）、AP、中继（Repeater）、桥接（Bridge）、客户端（Client） 五种

路由模式：拨号，DHCP，NAT

  • 自己进行 PPPoE 拨号（输入宽带账号密码）
  • 自己分配 IP 地址（192.168.x.x）
  • 自己做 NAT 转发、防火墙、DHCP
  • 自带 WiFi，插上网线就能用

AP 模式（无线接入点模式）

路由器只当无线发射器，不做路由功能。

  • 关闭 DHCP（不分配 IP）
  • 不拨号、不 NAT
  • 只提供 WiFi 信号 + 网线扩展
  • IP 由主路由 / 光猫分配

中继模式

不用网线，直接接收别人 WiFi 再放大。

  • 不用插网线
  • 接收主 WiFi → 放大发射
  • 名字和密码可以一样
  • 缺点：速度会减半，稳定性一般

桥接模式（WDS 桥接）
和中继类似，多路由器之间无线连接，扩大覆盖。
一般和中继功能合并在一起
适合：多路由组网

客户端模式（无线网卡模式）
路由器变成一个 “无线网卡”。
接收 WiFi，然后从 LAN 口输出网线给电脑 / 电视
适合：电视、台式机没有无线，用路由器接收 WiFi

名词解释：

拨号（PPPoE）：你家宽带要填的账号、密码。路由器 / 光猫拿着账号去运营商 “登录”。

DHCP：给手机、电脑发 IP 地址，子网划分。

NAT：作用：让多台设备共用一个宽带，流量转发，分流。

MAC 地址（物理地址）：网卡芯片里，出厂固化。格式：00:1A:2B:3C:4D:5E（16 进制，6 段）。作用：局域网内找设备。特点：全球唯一，不随网络、位置变化，属于数据链路层。

IP 地址（网络地址）：系统 / 路由器分配，ipv4,ipv6。作用：互联网通信，可动态 / 静态分配，换 WiFi、换网络就会变，属于网络层。

ISP服务商：网络业务提供商，电信，联通等。

拨号上网：电脑通过ISP账号上网
