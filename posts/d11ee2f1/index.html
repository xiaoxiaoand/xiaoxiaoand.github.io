搭建服务器

卸载nginx，3种方式

1.yum

yum remove nginx

2.make

需要在目录中make uninstall

如果MakeFile文件没有这个命令会失败

需要手动卸载这个文件夹

然后删除清理配置和日志文件

sudo rm -rf /etc/nginx
sudo rm -rf /var/log/nginx
sudo rm -rf /usr/local/nginx

3.rpm

安装nginx

Index of /download/

wget https://nginx.org/download/nginx-1.24.0.tar.gz

安装nginx-http-flv-module模块

这个应该要先下载git吧

git clone https://gitcode.com/gh_mirrors/ng/nginx-http-flv-module.git

一开始安装的是nginx-rtmp-module模块，利用nginx搭建rtmp流服务器 - ydqun - 博客园，封装了rtmp server。nginx可以用此模块可以很简单提供RTMP服务。

但是前端无法直接播放rtmp，然后需要nginx-http-flv-module模块，这个模块好像封装了nginx-rtmp-module

./configure \
—with-threads \
—with-http_stub_status_module \
—with-http_ssl_module \
—with-http_realip_module \
—with-stream \
—with-stream_ssl_module \
—add-module=/root/nginx-http-flv

./configure —add-module=/root/nginx-http-flv —with-http_ssl_module
make
make install

./configure —add-module=/root/nginx-http-flv-module
make
make install
