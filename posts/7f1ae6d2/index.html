Java概述

JDK（开发工具包）->JRE（运行环境）->JVM(Java虚拟机)+类库

通过JVM虚拟机实现跨平台

Java版本问题

1.JavaSE Java标准版

Java基础核心功能（语法，面向对象，集合，IO，多线程，网络，反射等），适用于桌面端的开发。

2.JavaME Java微型版

多用于嵌入式，移动端开发

3.JavaEE JAVA企业版

用于Web开发，包含Web请求接口规范等等。

B/S体系 ->浏览器/服务器

C/S体系 ->客户端/服务器

SSM框架

SSM框架是spring、spring MVC 、和mybatis框架的整合，是标准的MVC模式。标准的SSM框架有四层，分别是dao层（mapper），service层，controller层和View层。使用spring实现业务对象管理，使用spring MVC负责请求的转发和视图管理，mybatis作为数据对象的持久化引擎。

10.方法的访问级别：4种

静态

静态方法只能访问有静态修饰的成员,属性,方法.

最终态:无法被继承,必须有初始值.

java中一个文件，只能存在一个public类，并且名字要与文件名一致。

11.封装继承多态(oop)

this.成员变量

this.可以访问构造函数，但是只能在构造函数中使用，就是可以用一个构造函数访问另一个构造函数,但是只能在第一句

extends 继承

super.父类中的成员

子类构造方法会默认访问父类无参构造方法,且先执行父类构造方法

每个子类默认第一句都是调用父类构造方法.

方法重写

@Override

私有方法无法重写

多态

子转父

父类对象引用给到子类

父转子

抽象类 abstract

抽象方法没有方法体,并且抽象方法只能在抽线类中

抽象类无法实例化,继承的方法一定要重新父类的抽象方法.

12.包(package)

src目录下,没有包名称

package 打包

import 导入

13.接口 interface implement

继承接口的类必须实现接口所有的方法

接口中的成员变量:都是常量并且都是静态的(public static final a=10;)

接口中的方法:都是抽象方法

jdk8之后的接口中定义非抽象方法:default

接口没有构造方法

抽象类:对事物抽象

接口:对行为抽象
